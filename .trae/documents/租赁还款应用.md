# 租赁还款应用实现计划

## 技术栈选择
- **前端框架**: React + TypeScript + Vite
- **UI组件库**: Ant Design（提供丰富的表单、表格、通知组件）
- **状态管理**: React Context API
- **数据持久化**: localStorage
- **提醒功能**: 浏览器 Notification API

## 核心功能模块

### 1. 数据模型设计
- 租客信息（姓名、联系方式）
- 租赁记录（物品名称、数量、开始日期、归还日期）
- 费用信息（押金、租金、其他费用）
- 计租方式（按天/按月）
- 结算状态（已结清/未结清）

### 2. 主要页面
- **租赁列表页**: 显示所有租赁记录，未结清的排在前面
- **新增租赁页**: 表单录入租赁信息
- **租赁详情页**: 查看和编辑租赁记录
- **提醒设置页**: 配置提醒时间和方式

### 3. 核心功能实现
- 自动计算租金（按天/按月）
- 每日/每月提醒功能
- 押金和费用管理
- 归还标记和结算
- 数据本地持久化
