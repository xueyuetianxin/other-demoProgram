# 解决连接被拒绝错误

## 问题分析
开发服务器日志显示运行正常在 http://localhost:5174/，但浏览器显示 `ERR_CONNECTION_REFUSED`。

## 可能原因
1. **浏览器缓存** - 浏览器可能还在尝试连接旧的端口
2. **端口冲突** - 虽然服务器显示 5174，但可能有冲突
3. **网络问题** - 本地网络连接问题

## 解决方案
1. **强制刷新浏览器** - Ctrl + F5 清除缓存
2. **直接访问新URL** - 在新标签页打开 http://localhost:5174/
3. **清除浏览器缓存** - 使用无痕模式或清除缓存
4. **检查端口占用** - 如需要可以更换端口

## 执行步骤
1. 建议用户强制刷新浏览器（Ctrl + F5）
2. 如果无效，在新标签页直接访问 http://localhost:5174/
3. 如仍有问题，考虑更换端口或清除浏览器缓存
