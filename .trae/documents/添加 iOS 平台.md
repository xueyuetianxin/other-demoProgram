# 添加 iOS 平台

## 当前状态
✅ Android 平台已添加
✅ Web 资源已同步

## 执行步骤

### 1. 添加 iOS 平台
```bash
npx cap add ios
```

### 2. 同步 Web 资源
```bash
npx cap sync
```

### 3. 打开 iOS 项目
```bash
npx cap open ios
```

### 4. 使用 Xcode 打包
- 在 Xcode 中打开项目
- 配置应用信息
- 打包 IPA 文件

## 注意事项
- iOS 打包需要 Mac 电脑和 Xcode
- 需要 Apple Developer 账号
- 需要 iOS 设备或模拟器测试
