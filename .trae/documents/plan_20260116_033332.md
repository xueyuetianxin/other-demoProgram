# 修复 iOS 构建工作流

## 问题分析
GitHub Actions 在构建 iOS IPA 时，执行 `pod install` 失败，因为找不到 Podfile。项目使用的是 Swift Package Manager (SPM) 而不是 CocoaPods。

## 解决方案
修改 GitHub Actions 工作流，移除 CocoaPods 相关步骤，使用 Swift Package Manager 进行构建。

## 执行步骤
1. 更新 GitHub Actions 工作流文件，移除 CocoaPods 相关步骤
2. 更新构建命令，使用 .xcodeproj 而不是 .xcworkspace
3. 更新所有 Actions 到 v4 版本
4. 提交并推送更改
