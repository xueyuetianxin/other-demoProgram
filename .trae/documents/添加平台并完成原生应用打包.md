# 添加平台并完成原生应用打包

## 当前状态
✅ 项目已构建完成（dist 目录）
✅ Capacitor 配置完成
✅ Web 资源已同步

## 下一步操作

### 1. 添加 Android 平台
```bash
npx cap add android
```

### 2. 添加 iOS 平台（如果需要）
```bash
npx cap add ios
```

### 3. 打开项目
```bash
npx cap open android
npx cap open ios
```

### 4. 在 Android Studio / Xcode 中打包
- 使用 Android Studio 打包 APK/AAB
- 使用 Xcode 打包 IPA
