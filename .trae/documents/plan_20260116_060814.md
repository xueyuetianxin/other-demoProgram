# 修复 Node.js 版本不兼容问题

## 问题分析

错误信息：`The Capacitor CLI requires NodeJS >=22.0.0`

- Capacitor CLI 8.0.1 需要 Node.js 22.0.0 或更高版本
- 当前 GitHub Actions 工作流中使用的是 Node.js 20
- 版本不兼容导致 Capacitor 同步失败

## 解决方案

将 GitHub Actions 工作流中的 Node.js 版本从 20 升级到 22，以满足 Capacitor CLI 8.0.1 的要求。

## 执行步骤

1. 修改 `build-android.yml` 文件，将 Node.js 版本从 20 改为 22
2. 修改 `build-ios.yml` 文件，将 Node.js 版本从 20 改为 22
3. 提交并推送更改，验证修复效果
